## Programming & Tools

### The Missing Semester of Your CS Education
Covers programming skills that are rarely taught in class but are expected in many courses, research labs, and internships (Git, the command line, shell scripting, text editors, etc.).
- [Missing Semester Course](https://missing.csail.mit.edu/)

### Git
Git is the industry standard for version control and is assumed knowledge in many courses, research labs, and internships.
- [Git Tutorial](https://learngitbranching.js.org/?locale=en_US)
- [Missing Semester, Git Tutorial](https://missing.csail.mit.edu/2020/version-control/)

### Vim Resources
A handy reference if you decide to learn Vim or expect to work on remote Linux systems.
- [VIM Cheat Sheet Reference Guide](https://www.reddit.com/r/vim/comments/32r85c/this_is_my_favorite_vim_cheat_sheet_does_anyone/)

#### Scientific Computation

### Scientific Computation
Arguably the heart of Computational Engineering. I especially recommend Chapters 16, 17, 19, 20, and 21.
- [Python Programming And Numerical Methods: A Guide For Engineers And Scientists](https://pythonnumericalmethods.studentorg.berkeley.edu/notebooks/Index.html)

#### Programming Languages

### Python
The most widely used programming language in Computational Engineering for scientific computing, data analysis, and machine learning.
- [w3schools Python Tutorials](https://www.w3schools.com/python/default.asp)
- [GeeksForGeeks Python Tutorials](https://www.geeksforgeeks.org/python/python-programming-language-tutorial/)

### C++
Essential for high-performance scientific computing and many engineering software libraries.
- [w3schools C++ Tutorials](https://www.w3schools.com/cpp/default.asp)
- [GeeksForGeejs C++ Tutorials](https://www.geeksforgeeks.org/cpp/c-plus-plus/)

### MATLAB
Used heavily throughout engineering courses. MATLAB Academy also offers free interactive courses and certificates.
- [Mathworks MATLAB Resources](https://matlabacademy.mathworks.com/?page=1&sort=featured&s_tid=nav_learn_mlac)

#### IDEs

An Integrated Development Environment (IDE) combines a code editor, debugger, compiler/interpreter, and project management tools into a single application.

### JetBrains IDEs
Free for UT students with a registered eID. IntelliJ (Java), PyCharm (Python), and CLion (C++).
- [Jetbrains Student Account](https://www.jetbrains.com/academy/student-pack/?var=1)

### VS Code
My go-to editor. Lightweight, highly customizable, and supports virtually every programming language.
- [Visual Studio Code Download](https://code.visualstudio.com/)

### Google Colab
Run Python code directly in your browser without installing any software.
- [Google Colab](https://colab.research.google.com/)

## Mathematics

### Calculus Resources
Calculus is the foundation of engineering. This series develops the intuition behind the concepts rather than focusing only on computations.
- [3Blue1Brown's Calculus Series](https://youtube.com/playlist?list=PLZHQObOWTQDMsr9K-rj53DwVRMYO3t5Yr&si=YRFEfwC0QvrI1XJQ)
- [Calculus III (Multivariable Calculus) -- Dr. Trefor Bazett](https://youtube.com/playlist?list=PLHXZ9OQGMqxc_CvEy7xBKRQr6I214QJcd&si=SoLvw1mZiNgrxKlA)
- [Calculus IV (Vector Calculus) -- Dr. Trefor Bazett](https://youtube.com/playlist?list=PLHXZ9OQGMqxfW0GMqeUE1bLKaYor6kbHa&si=2iGE_nX8VW26zUQW)

### Linear Algebra Resources
Linear algebra appears everywhere in Computational Engineering. The resources below should help build intuition for matrices, eigenvalues, and linear transformations, and matrix factorization. 
- [3Blue1Brown's Linear Algebra Series](https://youtube.com/playlist?list=PLZHQObOWTQDPD3MizzM2xVFitgF8hE_ab&si=xPrW6Ct6L60HkQ21)
- [The Big Six Matrix Factorizations](https://nhigham.com/2022/05/18/the-big-six-matrix-factorizations/)
- [MathTheBeautiful's Linear Algebra Series](https://youtube.com/playlist?list=PLlXfTHzgMRUKXD88IdzS14F4NxAZudSmv&si=Va_IChr8vGfCynCH)
- [Short Post explaining Basic Linear Algebra Concepts](https://betterexplained.com/articles/linear-algebra-guide/)

### Logic Theory
Proofs are the language of mathematics. Understanding logical reasoning and quantifiers ("for all" and "there exists") will make upper-level mathematics much easier.
- [How to Read Logic -- Another Roof](https://youtu.be/JVjv0CCzHok?si=rGsIfcj0w6zkFvV7)
- [Discrete Math YouTube Playlist -- Dr. Trefor Bazett](https://youtube.com/playlist?list=PLHXZ9OQGMqxersk8fUxiUMSIx0DBqsKZS&si=6GVrZEkYhFmK5tzx)
- [Practice Problems on Logic Theory](https://people.iith.ac.in/aravind/Files-DM/dm-practiceProblems.pdf)
- [Great Advice on self-studying proofs](https://fearnomath.com/basic-proofs-sets-and-logic-curriculum-for-self-study/)

## Other

### Control Theory
One of the most important subjects in robotics, autonomous systems, optimization, and dynamical systems.
- [Control Bootcamp -- Steve Brunton](https://youtube.com/playlist?list=PLMrJAkhIeNNR20Mz-VpzgfQs5zrYi085m&si=T22od0HCqtYeq0H1)
- [Linear Systems Theory Book -- Hespanha](https://dl.poweren.ir/downloads/PowerEn/Book/2019/Apr/linear%20Systems%20Theory%20Hespanha%20%28PowerEn.ir%29%20.pdf)

### Andrew Ng's Machine Learning Course
One of the best machine learning courses available. A background in probability and statistics is recommended.
- [CS229 -- Andrew Ng (Stanford Engineering)](https://youtu.be/jGwO_UgTS7I?si=AuhLY8ug-uRNFZXk)

### Steve Brunton's Physics-Based Machine Learning Series
An excellent introduction to physics-informed neural networks (PINNs).
- [Physics Informed Machine Learning -- Steve Brunton](https://www.youtube.com/playlist?list=PLMrJAkhIeNNQ0BaKuBKY43k4xMo6NSbBa)

### Optimization
A gentle introduction to optimization algorithms and why they matter in machine learning and engineering.
- [Algorithms for Optimization -- Mykel J. Kochenderfer](https://algorithmsbook.com/optimization/)
- [Numerical Optimization -- Jorge Nocedal](https://www.ime.unicamp.br/~pulino/MT404/TextosOnline/NocedalJ.pdf)

### LaTeX Materials
A quick reference for writing mathematical documents in LaTeX.
- [LaTeX Quick Reference](https://quickref.me/latex.html)
- [Overleaf](https://www.overleaf.com/project)
